# 量子核心 - 无人机集群防御

一款结合了无人机集群编程控制与动态战场交互的科幻极简风策略塔防游戏。

## 游戏特色

### 1. 无人机集群系统
- **三种无人机类型**：
  - **侦察型**：移动速度快，适合快速部署和侦察，能量消耗低（20能量）
  - **战斗型**：伤害高、生命值高，是战斗主力，能量消耗中等（50能量）
  - **支援型**：可以自动治疗友军，适合持久战，能量消耗低（30能量）

- **四种编队形态**：
  - **分散**：无人机以中心点向四周散开，适合范围防御
  - **线形**：无人机排列成直线，适合正面推进
  - **圆形**：无人机组成圆形防御阵型，适合保护核心区域
  - **方阵**：无人机组成紧密方阵，适合集中火力

### 2. 经济循环系统
- **双重资源体系**：
  - **能量**：用于制造无人机和释放部分技能，通过击退敌人获取
  - **晶石**：用于释放高级技能，通过破坏地图上的能量晶石获取

- **动态资源生成**：
  - 击杀敌人获得能量奖励（每个敌人15能量）
  - 破坏能量晶石矿脉获得晶石资源（20-40晶石/个）
  - 完成波次获得额外能量奖励（随波次递增）

### 3. 编程控制台
支持基础无人机集群编程指令：

```
move <x> <y>      - 移动整个集群到指定网格坐标
attack            - 集群进入攻击模式
formation <type>  - 切换编队形态 (spread/line/circle/phalanx)
scout             - 侦察型无人机开始巡逻
```

示例：
```
move 8 6
formation circle
attack
```

### 4. 指挥官技能系统
四种全局干预技能，可在关键时刻扭转战局：

| 技能名称 | 消耗 | 效果 | 冷却时间 |
|---------|------|------|---------|
| **轨道激光** | 100晶石 | 对目标区域150半径内所有单位造成100点伤害 | 30秒 |
| **电磁脉冲** | 150能量 | 使目标区域200半径内所有敌人瘫痪3秒 | 20秒 |
| **量子护盾** | 80能量 | 为所有无人机激活护盾，持续5秒免疫伤害 | 25秒 |
| **紧急增援** | 200晶石 | 立即在地图中心召唤3架战斗型无人机 | 40秒 |

### 5. 视觉效果
- **科幻极简风格**：深色背景配合霓虹灯光效，营造未来科技感
- **粒子特效系统**：
  - 无人机部署特效
  - 攻击弹道和命中效果
  - 治疗和护盾特效
  - 爆炸和技能释放动画
- **网格系统**：清晰的网格参考线，便于定位和部署

## 游戏玩法

### 基本操作
1. **选择无人机类型**：点击左侧面板中的无人机类型按钮
2. **部署无人机**：在战场上点击想要放置的位置
3. **开始波次**：准备好后点击右侧面板的"开始波次"按钮
4. **调整编队**：使用编队形态按钮调整无人机阵型
5. **释放技能**：选择技能后点击战场目标位置释放

### 策略建议
1. **前期策略**：多部署侦察型无人机，利用其高机动性快速占据有利位置
2. **中期策略**：增加战斗型无人机数量，开始开采能量晶石
3. **后期策略**：配合支援型无人机和技能系统，应对越来越强的敌人波次
4. **技能时机**：电磁脉冲适合在敌人密集时使用，量子护盾适合在关键时刻保护集群

### 编程技巧
- 使用 `move` 指令可以让整个集群精确移动到指定位置
- `formation` 配合 `move` 可以实现精准的战术部署
- `scout` 指令可以让侦察型无人机单独执行巡逻任务

## 技术架构

### 前端技术栈
- **HTML5 Canvas**：游戏渲染核心，支持高性能2D绘图
- **原生JavaScript**：游戏逻辑实现，无需任何框架依赖
- **CSS3**：UI样式和动画效果，使用渐变和阴影营造科幻感

### 系统架构
```
├── index.html          # 主页面结构
├── style.css           # 样式文件
└── game.js             # 游戏核心逻辑
    ├── Game 类         # 游戏主控制器
    ├── 游戏循环系统    # update + render
    ├── 单位管理系统    # 无人机/敌人/晶石
    ├── 技能系统        # 指挥官技能
    ├── 粒子系统        # 视觉特效
    └── UI 交互系统     # 事件处理
```

### 游戏循环
```
requestAnimationFrame(gameLoop)
    ├── update(deltaTime)  # 游戏状态更新
    │   ├── 更新技能冷却
    │   ├── 更新无人机状态
    │   ├── 更新敌人状态
    │   ├── 更新弹道
    │   ├── 更新粒子
    │   ├── 碰撞检测
    │   └── 波次管理
    └── render()           # 渲染画面
        ├── 绘制网格
        ├── 绘制晶石
        ├── 绘制无人机
        ├── 绘制敌人
        ├── 绘制弹道
        ├── 绘制粒子
        └── 绘制UI提示
```
